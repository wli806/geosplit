# PRD 文件夹

每个开发 **slice(纵切片)** 写一份 PRD,文件名 `slice-X-Y.md`。
- 一个 slice = 一个**端到端可验证**的小目标(从输入到前端能看到结果)。
- PRD 写清:目标、范围(IN/OUT)、测试地块、输入输出契约、做哪几步、**验收标准**、明确不做的、待 planner 确认的。
- 架构总纲见 `../Records/项目进展与决策.md`;数据接口见 `../Records/Auckland-GIS-数据接口手册.md`。
